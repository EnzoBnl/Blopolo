# Blopolo
## Overview
Jumping Blop Game powered by GPong library (github.com/enzobnl/gpong)

## Features
- Sick graphics
- Super Jumps
- Falling Blocks

## Goal
Try to survive as long as possible !  

## Test it
**Requires:** 
- Windows

 1. Run `Lancer.exe`
 2. Play !
